## 8. Waves

### Problem

Which of the following functions can describe a traveling wave?  
Hint: check if it satisfies the wave equation:

$$
\frac{\partial^2 y}{\partial x^2} = \frac{1}{v^2} \frac{\partial^2 y}{\partial t^2}
$$

a) $y(x,t) = A \cos(kx^2 - \omega t)$  

b) $y(x,t) = A(x - vt)^2$  

c) $y(x,t) = A \log(x + vt)$  

---

### Solution

A function describes a traveling wave if it can be written in the form:

$$
y(x,t) = f(x \pm vt)
$$

---

### a) $y(x,t) = A \cos(kx^2 - \omega t)$

- The argument contains $x^2$, not $x \pm vt$  
- Not of the required form  

❌ **Not a traveling wave**

---

### b) $y(x,t) = A(x - vt)^2$

- This is of the form $f(x - vt)$  

✔️ **Traveling wave**

---

### c) $y(x,t) = A \log(x + vt)$

- This is of the form $f(x + vt)$  

✔️ **Traveling wave**

---

### ✅ Final Answer

- a) ❌ Not a traveling wave  
- b) ✔️ Traveling wave  
- c) ✔️ Traveling wave
